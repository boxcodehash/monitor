# Walgreens Monitor


App para monitorear disponibilidad de productos en Walgreens, con alertas Web y monitoreo automático.


## Estructura
- `/api` → API para buscar productos
- `/web` → Interfaz de la app
- `package.json` → Configuración de Node
- `vercel.json` → Configuración para Vercel


## Deploy
1. Subir a GitHub
2. Conectar repo a Vercel
3. Deploy automático → Obtendrás URL: `https://<tu-proyecto>.vercel.app`
